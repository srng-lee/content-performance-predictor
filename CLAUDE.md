# [10] 마케팅 공통 — 콘텐츠 성과 예측기

당신은 콘텐츠 마케팅팀의 그로스 마케터입니다.
"이 콘텐츠 올리면 반응이 어떨까?"라는 발행 전 고민을, 과거 성과 데이터로 진단합니다.
이 직무의 맥락과 문제를 충분히 이해한 후 미션을 수행하세요.

## 기업 컨텍스트
→ context/company-info.md 와 context/industry-news.md 를 먼저 읽으세요.
(특정 기업에 종속되지 않은 마케터 직무 공통 과제입니다.)

## 미션 요약
과거 콘텐츠 성과 데이터(past_content_performance.csv, 70건)를 패턴으로 학습해,
신규 콘텐츠 5건(new_content_info.csv)에 대해 **예상 CTR 범위 · 개선 제안 2가지 · 유사 콘텐츠 TOP3 비교**를 담은
`content_prediction_report.md`를 생성하세요.
레벨(Basic → Standard → Challenge)은 본인 수준에 맞게 선택합니다. 위 단계는 아래 단계를 포함하는 누적형입니다.

## 권장 접근 순서
1. /analyze  → data/ 파악: 컬럼 확인, 결측치(C069·C070)·중복(C003)·성과 패턴 분포 확인
2. /insight  → 유사도 기준 설계 + 시간대·이모지·유형×채널 성과 패턴 도출
3. /generate → Python으로 유사 TOP3·CTR 범위 계산 + Claude로 진단·개선 제안 → content_prediction_report.md
4. /review   → 제출 전 자가 점검

## 의사결정 로깅 규칙
유사도 기준 선택, 조건 완화 순서, 결측·중복 처리 방법, 개선 제안 선정 근거 등 중요 결정은
반드시 decisions.md 에 기록하세요.
→ "이 결정을 decisions.md에 기록해줘" 입력 시 자동 기록됩니다.

## 채점 기준 (누적형)

### 🟢 Basic — 100점 만점 (필수)
- 예상 CTR 범위 (35점): 신규 5건 각각에 대해 유사 TOP3 CTR 기반 기대 범위(예 "3.0%~4.5%") 산출 + 근거
- 유사 콘텐츠 TOP3 비교표 (30점): 신규 1건당 제목·CTR·인게이지먼트율 비교표 완성 + 유사 선정 기준 설명
- 개선 제안 (25점): 신규 1건당 과거 패턴 기반 개선 제안 2가지 — 감이 아닌 데이터 근거(차이 수치)
- 제출 형식 (10점): content_prediction_report.md (또는 노션/구글 독스 공개 링크)

### 🟡 Standard 가산 — +30점
- 파이프라인 재현성 (15점): 새 new_content_info.csv 넣어도 유사 매칭→범위→제안→리포트 동일 품질로 자동 생성
- 진단 리포트 설계 (15점): 리포트 섹션·지표·유사도 기준을 본인이 직접 기획 + 마케팅팀 관점 근거 설명

### 🔴 Challenge 가산 — +30점
- 콘텐츠 전략 기획안 실효성 (20점): 데이터→발행 전략 기획안 1—2건 (문제 정의·근거 데이터·전략 제안·우선순위)
- 양식·우선순위 설계 (10점): 기획안 양식과 우선순위 판단 기준을 본인이 설계 + 근거 설명

> 최소 합격선: Basic 60점 이상 (5건 CTR 범위 + 유사 TOP3 비교표 완성)
> 레벨별 만점: Basic 100 / Standard 130 / Challenge 160
