진단 5건에서 반복된 손실 요인(점심·아침 발행, 이모지 누락, 과도한 제목 길이)은 단발 수정이 아니라 콘텐츠 캘린더 전반의 규칙으로 바꿔야 누적 효과가 난다. CTR 상승폭·적용 범위·실행 난이도 3축으로 점수화해 우선순위를 정했다.

기획안 01
저녁 발행 + 이모지 제목을 콘텐츠 캘린더 기본값으로
소셜(인스타·유튜브) 콘텐츠의 발행 시간·제목 규칙을 데이터 최적값으로 표준화
Priority
P0
문제 정의
진단 5건 중 ②③⑤가 점심·아침 발행 또는 이모지 누락으로 같은 조건 최적군 대비 CTR이 낮게 예측됨. 학습셋에서도 점심·아침·이모지 없음 콘텐츠가 저CTR 구간에 몰려 있음 — 개별 콘텐츠 문제가 아닌 발행 규칙의 문제.
전략 제안
• 소셜 콘텐츠 기본 발행 시간을 18~20시로 고정
• 인스타·유튜브 제목에 이모지 1개 + 숫자/결과형 키워드 가이드 의무화
• 콘텐츠 캘린더 템플릿에 시간·이모지 체크박스 내장
근거 데이터
시간대저녁(18·20) 평균 CTR이 아침(8) 대비 약 1.4%p 높음 (학습 69건)
이모지인스타 동일 조건 이모지 vs 없음 약 1.1%p 차이 (예: C007 4.6% vs C019 3.4%)
진단신규 ⑤ 이모지+저녁 적용 시 3.0~3.6% → 4%대 진입 여지

기획안 02
블로그·인포그래픽을 숏폼/카드뉴스로 리포맷하는 멀티 발행
CTR 상한이 낮은 포맷의 핵심을 고CTR 포맷으로 재가공해 인스타 저녁 동시 발행
Priority
P1
문제 정의
진단 ③(블로그·트렌드)·④(인포그래픽·후기)는 포맷 특성상 CTR 상한이 낮음(2.8~4.0%, 3.2~3.4%). 콘텐츠 자체는 좋지만 도달·클릭이 포맷에 갇혀 있음. 학습셋에서 같은 주제라도 숏폼·카드뉴스 포맷이 더 높은 CTR.
전략 제안
• 블로그 트렌드 글 → 핵심 3포인트 카드뉴스로 요약, 인스타 저녁 발행
• 인포그래픽 후기 → 30초 숏폼으로 리포맷(수치 1개 후킹)
• 원본은 SEO 유입, 리포맷본은 소셜 클릭 — 채널별 역할 분리
근거 데이터
③블로그 트렌드 2.8~4.0% vs 인스타 트렌드 카드뉴스 군 4.0%+
④인포그래픽 후기 3.2~3.4% vs 후기 숏폼(C056) 4.2%
패턴숏폼+유튜브/인스타가 학습셋 최상위 CTR 그룹
