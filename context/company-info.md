# 마케팅 공통 — 콘텐츠 성과 예측 도메인 컨텍스트

> 이 과제는 특정 기업에 종속되지 않은 **마케터 직무 공통 과제**입니다.
> "이 콘텐츠 올리면 반응이 어떨까?"라는 마케터의 매일의 고민을, 과거 성과 데이터를 근거로 진단하는 흐름을 다룹니다.

## 이 과제가 다루는 일

콘텐츠 마케터는 카드뉴스·숏폼·블로그·인포그래픽을 매일 기획·발행합니다. 발행 직전 늘 같은 질문에 부딪힙니다.

- "이 제목, 이 시간대에 올리면 클릭률이 잘 나올까?"
- "지난번 비슷한 콘텐츠는 반응이 어땠더라?"
- "발행 전에 방향이 맞는지 확인할 방법이 없을까?"

대부분은 **감(感)** 으로 결정하고, 발행 후에야 성과를 확인합니다. 과거 데이터가 있어도 매번 수동으로 비슷한 콘텐츠를 찾아 비교하기는 번거롭습니다. 이 과제는 그 과정을 **과거 성과 데이터 기반 진단**으로 바꿉니다.

## 콘텐츠 성과 핵심 지표

| 지표 | 정의 | 비고 |
|------|------|------|
| **CTR (클릭률)** | 콘텐츠를 본 사람 중 클릭한 비율 `클릭수 / 노출수 × 100` | 이 과제의 핵심 예측 대상 |
| **engagement_rate (인게이지먼트율)** | (좋아요+댓글+공유) / 도달수 × 100 | 콘텐츠 공감·확산도 지표 |
| **reach (도달수)** | 콘텐츠가 노출된 순 사용자 수 | 절대 규모 지표 |
| **posting_hour (발행 시간)** | 8 / 12 / 18 / 20시 | 시간대별 트래픽 차이 큼 |
| **headline_length (제목 글자수)** | 제목의 글자 길이 | 너무 길면 모바일에서 잘림 |
| **has_emoji (이모지 여부)** | 제목에 이모지·감정 표현 포함 여부 | 소셜에서 클릭 유인 |

## 콘텐츠 유형 (type)

| 유형 | 특징 | 강한 채널 |
|------|------|-----------|
| **카드뉴스** | 슬라이드형 정보 요약. 저장·공유 유도 | 인스타그램 |
| **숏폼** | 15~60초 영상. 짧고 강한 후킹 | 유튜브·인스타그램 |
| **블로그** | 긴 글. 검색 유입(SEO)에 강함 | 블로그(네이버) |
| **인포그래픽** | 데이터·통계 시각화 한 장 | 인스타그램·블로그 |

## 주제 카테고리 (topic_category)

| 카테고리 | 설명 |
|----------|------|
| **생산성** | 업무 효율·툴·루틴. 저장·실용 콘텐츠 |
| **커리어** | 취업·이직·연봉·직무 역량. 검색·정보 수요 높음 |
| **트렌드** | 시장·AI·채용 동향. 시의성 높을 때 폭발 |
| **후기** | 수강·이직·경험담. 진정성·인게이지먼트 강세 |

## 채널 (channel)

| 채널 | 사용자 특성 | CTR 경향 |
|------|-------------|----------|
| **인스타그램** | 이미지·이모지·해시태그 민감. 저녁 트래픽 피크 | 카드뉴스·숏폼에서 강세 |
| **블로그** | 검색 유입 중심. 제목 키워드가 핵심 | CTR 안정적, 인게이지먼트는 낮은 편 |
| **유튜브** | 썸네일·제목 후킹. 저녁·주말 강세 | 숏폼에서 CTR 최고 |

## 성과에 영향을 주는 요인 (데이터에 심긴 경향)

> 아래는 제공 데이터(`past_content_performance.csv`)에 실제로 반영된 경향입니다. 참여자가 직접 발견·검증해야 합니다.

1. **발행 시간대**: 저녁(18·20시) > 점심(12시) > 아침(8시). 저녁 발행이 CTR 상위에 집중.
2. **이모지**: `has_emoji=True`가 인스타그램·유튜브에서 CTR을 끌어올림.
3. **유형×채널 궁합**: 숏폼+유튜브, 카드뉴스+인스타그램 조합이 고CTR 구간.
4. **제목 길이**: 블로그 검색형은 길어도 OK, 소셜에서 제목이 너무 길면(30자 이상) CTR 하락.
5. **블로그의 역설**: 블로그는 CTR은 중간이어도 후기·커리어 주제에서 인게이지먼트가 상대적으로 높음.

## 왜 "예측"이 아니라 "패턴 기반 진단"인가

| ML/통계 예측 모델 | 패턴 기반 진단 (이 과제의 방식) |
|-------------------|----------------------------------|
| 수백~수천 건으로 학습 | 과거 70건에서 **유사 콘텐츠**를 찾아 실제 성과를 보여줌 |
| 다변수 회귀·신경망 | 유사도 계산 + CTR 정렬 (Python) |
| 새 입력 → 단일 예측값 | "유사 TOP3의 실제 CTR이 3.0~4.5%였다" 같은 **참고 범위 + 근거** |
| **70행으로는 과적합·신뢰구간 과대로 불가** | 70행으로도 충분히 유용한 의사결정 근거 제공 |

핵심: 신규 콘텐츠 1개를 받아 **과거 유사 콘텐츠 TOP3**를 찾고, 그 실제 CTR로 **기대 범위**를 제시하며, 상위 성과 콘텐츠와의 차이에서 **개선 제안**을 도출합니다.

## 유사 콘텐츠 TOP3 찾는 방법 (가이드)

1. 신규 콘텐츠 조건 정의: `type` + `topic_category` + `channel` + `posting_hour` + `has_emoji`
2. 동일 조건 필터링 → 결과가 부족하면 조건을 하나씩 완화(예: posting_hour → topic_category → channel 순)
3. 유사도 점수로 정렬(완전 일치 가중치 부여) → 상위 3개 추출
4. 상위 3개의 CTR 최소~최대를 **예상 CTR 범위**로 제시

## 개선 제안 방법론

- "이렇게 하면 좋을 것 같습니다"(감)가 아니라
- "유사 조건에서 이 요소가 다를 때 CTR이 X%p 차이났습니다"(근거)

**제안 예시**
- "저녁(20시) 발행으로 변경 시: 같은 유형·채널의 저녁 발행 평균 CTR 4.8% (현재 점심 발행 평균 3.4%)"
- "제목에 이모지 추가 시: 같은 채널 이모지 포함 평균 CTR 4.2% (미포함 3.0%)"

## 참고 용어 사전

| 용어 | 정의 |
|------|------|
| **CTR** | Click-Through Rate. 노출 대비 클릭 비율(%). 콘텐츠 후킹·매력도 핵심 지표. |
| **engagement_rate** | 인게이지먼트율. (좋아요+댓글+공유)/도달수 × 100. 공감·확산도. |
| **reach (도달)** | 콘텐츠가 노출된 순 사용자 수. CTR의 분모가 되는 노출과는 구분(여기선 규모 지표로 사용). |
| **숏폼** | 15~60초 세로형 짧은 영상. 릴스·쇼츠 등. |
| **카드뉴스** | 여러 장 슬라이드로 정보를 요약한 이미지 콘텐츠. |
| **인포그래픽** | 데이터·통계를 한 장 이미지로 시각화한 콘텐츠. |
| **A/B 테스트** | 두 버전(A/B)을 동시 노출해 성과를 비교하는 실험. 이 과제는 발행 **전** 방향 검증을 다룸. |
| **벤치마크** | 비교 기준이 되는 과거·업계 성과 수치. |
| **유사도** | 신규 콘텐츠와 과거 콘텐츠가 조건(유형·주제·채널·시간·이모지)에서 얼마나 일치하는지의 정도. |
