> 감이 아니라 데이터로, 발행 전략까지 자동 도출  
> 스파르타 AX 해커톤 2026 · 마케팅 공통 문제 #10 · 이나경

---

## 📌 한 줄 요약

신규 콘텐츠 정보를 입력하면, 과거 성과 데이터 패턴을 근거로 **예상 CTR 범위 · 개선 제안 · 유사 TOP3**를 자동으로 진단하는 파이프라인

| **내가 재정의한 문제** | '더 정교한 예측 모델'이 아니라 **'있는 기준을 자동으로 적용하는 도구'** 가 필요하다 |
| **해결 방향** | 파이프라인 자체가 그 수단 — Claude Code와 대화하며 설계·구현 |

## 🛠️ 개발 과정

1. **Claude Code와 대화하며 설계** — 유사도 기준·가중치를 대화로 정하고 즉시 코드로 반영

4. **새 CSV로 재현성 테스트** — 다른 데이터를 넣어도 동일 품질로 재현되는지 확인

## 💡 핵심 의사결정

### 1. 유사도 기준 설계
- **문제:** "5개 요소로 유사도를 정하라"는 재료만 있었고, 조합 방식은 없었다
- **행동:** 형식이 다르면 비교 자체가 무의미 → type·channel은 절대기준, 이모지는 개인취향이라 topic보다 낮은 가중치(4:2:1)로 설계
- **결과:** 신규 5건 모두 하드필터 통과, 후보 6~22건씩 확보

### 2. 모든 숫자는 실측으로 검증
- **문제:** 시간대 효과(+1.48%p) + 이모지 효과(+1.06%p) = +2.54%p라고 가정
- **행동:** 핵심 수치는 반드시 원본 데이터로 직접 재현해 검증한다는 원칙 수립
- **결과:** 이론상 2.54%p → **실측 상호작용 효과 0.99%p로 정정** (두 효과가 겹침을 발견)
