# [10] 마케팅 공통 — 콘텐츠 성과 예측기

트랙: 마케팅 (직무 공통) | 작업 시간: 3시간 | 개인 과제
채용 TO: 콘텐츠 마케터 / 그로스 마케터

## 기업 배경

이 과제는 특정 기업에 종속되지 않은 **마케터 직무 공통 과제**입니다. 카드뉴스·숏폼·블로그·인포그래픽을 매일 기획·발행하는 콘텐츠 마케터라면 회사·도메인과 무관하게 마주치는 문제를 다룹니다. 인스타그램·블로그·유튜브 등 여러 채널에 콘텐츠를 발행하고, CTR(클릭률)·인게이지먼트율로 성과를 측정하는 환경을 가정합니다.

## 현재 상황과 문제

마케터는 콘텐츠 발행 직전 늘 "이거 올리면 반응이 어떨까?"를 고민합니다. 그러나:

- 발행 전 반응 예측을 **감(感)** 에 의존합니다.
- 과거 성과 데이터가 있어도 **패턴을 분석해 적용하기가 번거롭습니다.**
- A/B 테스트는 발행 **후** 비교 — 발행 전 방향 검증 수단이 없어 **실패할 콘텐츠를 그대로 발행**합니다.
- 유사 콘텐츠를 찾아 벤치마크하는 일을 **매번 수동**으로 합니다.

결과적으로 저성과가 예상되는 콘텐츠를 거르지 못하고, 시간대·포맷·제목 선택이 데이터 근거 없이 반복됩니다.

## 미션

신규 콘텐츠 정보를 입력받아, 과거 성과 데이터 패턴을 근거로:
1. **Python으로 유사 콘텐츠 TOP3를 계산**하세요 (유사도 + 성과 정렬)
2. **Claude로 예상 성과 범위를 진단**하고 개선 제안을 도출하세요
3. 결과를 `content_prediction_report.md` 로 정리하세요 — 신규 콘텐츠 1건당:
   - **예상 CTR 범위** (유사 TOP3 CTR 기반, 예: "3.0%~4.5%")
   - **개선 제안 2가지** (과거 패턴 기반, 데이터 근거 포함)
   - **유사 콘텐츠 TOP3 비교표** (제목 · CTR · 인게이지먼트율)

> ⚠️ 이것은 **ML 예측 모델 과제가 아닙니다.** 70행으론 ML이 불가능하므로, 유사 콘텐츠의 **실제 성과**로 기대 범위를 제시하는 **패턴 기반 진단**입니다.

## 입력 데이터

**data/past_content_performance.csv — 70건의 과거 콘텐츠 성과**
컬럼: `content_id, title, type, topic_category, channel, ctr, engagement_rate, reach, headline_length, has_emoji, posting_hour`
- type: 카드뉴스 / 숏폼 / 블로그 / 인포그래픽
- topic_category: 생산성 / 커리어 / 트렌드 / 후기
- channel: 인스타그램 / 블로그 / 유튜브
- posting_hour: 8 / 12 / 18 / 20

**data/new_content_info.csv — 5건의 예측 대상 신규 콘텐츠**
컬럼: `title, type, topic_category, channel, headline_length, has_emoji, posting_hour`
(성과 컬럼 없음 — 이것을 진단하는 것이 미션)

데이터 품질 주의사항 (정제가 필요합니다):
- **결측 2건**: C069는 engagement_rate가, C070은 reach가 빈 셀 → 유사 비교 시 처리 기준 필요
- **완전 중복 1건**: content_id `C003`이 동일 내용으로 2번 등장 → 패턴 계산 전 제거 권장 (제거 시 69건)
- has_emoji는 `True`/`False` 문자열 — 불리언 처리 주의

## 제출 가이드

**기본 (필수)**
- 신규 5건 진단이 담긴 `content_prediction_report.md` 1개
- output/ 폴더에 저장

**권장**
- 유사 매칭·범위 계산 Python 스크립트 (새 new_content_info.csv에도 재현)
- HTML 시각화 리포트 (유사 TOP3 비교 + 게이지)

**도전**
- 데이터 기반 콘텐츠 발행 전략 기획안 (우선순위 포함)
- Skill(/명령)로 패키징하여 재현 가능한 형태로 제출
