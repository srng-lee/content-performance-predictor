# /review — 제출 전 자가 점검

생성한 `content_prediction_report.md`를 채점 기준에 비춰 자가 점검하라. 항목별 ✅/⚠️/❌ 와 근거를 제시.

## 🟢 Basic (100점)
- [ ] 예상 CTR 범위 (35점): 신규 **5건 전부** 유사 TOP3 기반 범위 + 근거가 있는가
- [ ] 유사 TOP3 비교표 (30점): 5건 각 제목·CTR·인게이지먼트율 표 완성 + 유사 선정 기준 설명
- [ ] 개선 제안 (25점): 1건당 2가지, **데이터 근거(차이 수치)** 포함 (감·일반론 아님)
- [ ] 제출 형식 (10점): content_prediction_report.md 또는 공개 링크

## 🟡 Standard (+30점)
- [ ] 파이프라인 재현성 (15점): 새 new_content_info.csv 넣어도 동일 품질로 자동 생성되는가
- [ ] 진단 리포트 설계 (15점): 섹션·지표·유사도 기준을 본인이 기획 + 근거 설명

## 🔴 Challenge (+30점)
- [ ] 전략 기획안 실효성 (20점): 문제·근거 데이터·전략·우선순위가 있는 기획안 1~2건
- [ ] 양식·우선순위 설계 (10점): 기획안 양식·판단 기준을 본인이 설계 + 근거

## 데이터 처리 점검
- [ ] 중복 C003 제거(69건 기준) 명시했는가
- [ ] 결측 C069·C070 처리 방침을 적었는가
- [ ] has_emoji 문자열→불리언 처리했는가

## 출력
- 항목별 점검 결과 + 예상 점수
- 미흡 항목 보완 가이드
- decisions.md 기록 누락분 점검
